# TweenSystem
Tween Studio is a visual editor for Roblox TweenService that lets developers create complex UI animations using a timeline, live preview, and drag-and-drop controls. It generates clean Lua code for sequences, easing, and multi-property tweens, removing repetitive scripting.

Tween Studio — Description
Tween Studio is a visual animation editor for Roblox UI that allows developers to design, preview, and export complex Roblox TweenService animations without writing boilerplate code.
Instead of manually scripting tweens, Tween Studio provides a timeline-based interface where animations can be constructed using drag-and-drop blocks, edited through a property panel, and tested in real time. Each animation sequence can then be exported as clean, production-ready Lua code.

Core Capabilities

Visual Timeline Editor
Build animations using a structured timeline. Arrange tweens sequentially or in parallel, adjust durations, and control timing precisely.
Multi-Property Tweens
Animate multiple UI properties within a single tween
Position (UDim2)
Size
Rotation
Transparency
Color (BackgroundColor3)
Text/Image transparency

Advanced Tween Configuration
Full control over:
Easing styles and directions
Delays and repeat counts
Reversing behavior

Live Preview System
Instantly simulate animations on UI elements to validate motion, timing, and transitions before exporting.

Code Generation Engine
Automatically generates optimized Lua scripts using TweenService, including:
Proper TweenInfo construction
Multi-property tween handling
Sequencing and parallel execution logic

Intended Use Cases
Rapid prototyping of UI animations
Eliminating repetitive tween scripting
Teaching animation concepts in Roblox development
Creating reusable animation patterns for games

Positioning
Tween Studio functions as a bridge between visual design and scripting, similar to how video editors abstract timelines while still producing structured output. It is designed for both:
Beginners who want to avoid low-level coding
Advanced developers who want to accelerate workflow
